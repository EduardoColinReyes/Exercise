## Ejercicios
 
### Temperaturas Árticas — Fun with Python Built-in Math Functions
 
Análisis de datos de temperatura de la Antártida usando funciones matemáticas nativas de Python.
 
**Lo que aprenderás:**
- `max()` y `min()` → temperatura más alta y más baja del conjunto de datos
- `sum()` y `len()` → cálculo de la temperatura media
- `abs()` → distancia bajo cero de la temperatura más fría
---
 
### Conversiones de Texto — Python Built-in Conversion Functions
 
Exploración de funciones de conversión de tipos de datos.
 
**Lo que aprenderás:**
- `type()` → inspección del tipo de dato de una variable
- `int()` → conversión de cadena a entero
- `str()` → conversión de entero a cadena
